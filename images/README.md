# Images

Assets utilisés sur le site :

- `livingstone_nav.png` — logo horizontal, fond transparent (1200×420). Utilisé en navigation et en footer sur toutes les pages publiques (index, hub cas-pratiques, articles, pages légales).
- `livingstone_carre.png` — logo carré, fond noir (1080×1080). Utilisé en favicon, en `og:image`, en `JSON-LD publisher.logo` et en `JSON-LD article.image`.
- `mikael-portrait-web.jpg` — portrait du fondateur, version web compressée (2000 px, JPEG q80, ~124 Ko). **Référencée par `index.html`** (sections Hero & Portrait).
- `bureau-web.jpg` — photo des bureaux, version web compressée (2000 px, JPEG q80, ~236 Ko). **Référencée par `index.html`** (section Protection luxembourgeoise).
- `mikael-portrait.jpg` / `bureau.jpg` — **originaux haute résolution** (8000+ px, ~15 Mo / ~24 Mo) conservés comme masters, **non servis sur le site**. Regénérer les versions `-web.jpg` à partir de ces fichiers si besoin (sharp : `resize 2000px`, `jpeg quality 80`).
